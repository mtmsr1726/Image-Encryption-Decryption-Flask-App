# Image Encryption & Decryption Flask App

A secure image sharing application built using Flask, Python, OpenCV, and SQLite.

## Features
- User Login and Registration
- Image Upload
- Image Encryption
- Image Decryption
- Key Verification
- Image History Tracking

## Technologies Used
- Python
- Flask
- OpenCV
- NumPy
- SQLite
- HTML, CSS

## Run Locally

```bash
pip install flask numpy opencv-python werkzeug
python app.py
```
